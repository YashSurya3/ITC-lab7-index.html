Creative Mad Libs











Project Notes

Backend Configuration

An Express.js server powers this app, enabling static file hosting and form data handling. Middleware like express.urlencoded was used to extract form data, and morgan aids in logging requests.

Interface Elements

The form features five inputs—noun, verb, adjective, adverb, and location—all required to ensure full interaction.

Data Processing

The /ITC505/lab-7/index.html POST endpoint verifies all entries before producing a personalized story. Missing data prompts users to return and complete the form.

Styling and Presentation

CSS was used to create a clean, responsive layout that highlights both form usability and readability.

Testing and Validation

The system was evaluated for correct form operation, response handling, and display of the last modification date.
